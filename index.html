Feature Attribution Table
Dominant Drivers

rent_arrears_balance (0.312) and days_in_arrears (0.241) together explain ~55% of model output variance. These are direct arrears signals — model is using the right features.

Protective Factors

payment_freq_30d, contact_responsiveness and tenure_months all reduce predicted risk. Long-tenure responsive tenants are correctly penalised less even when balance dips.

Fairness Check

uc_claim_active (0.187) present but lower than arrears signals. Fairlearn post-hoc optimizer ensures UC status does not drive threshold discrimination vs non-UC with same arrears balance.

⓵ L1 — Base Models (Heterogeneous Ensemble) 3 MODELS
XGBoost 2.0.3 40% WEIGHTPRIMARY
n_estimators=800 · max_depth=6 · learning_rate=0.05 · subsample=0.8 · colsample_bytree=0.75 · reg_alpha=0.1 · reg_lambda=1.5 · tree_method='hist' · booster='dart' · rate_drop=0.1
Primary arrears classifier. DART dropout prevents over-reliance on early trees. L1+L2 regularisation reduces overfitting on sparse UC benefit features. Monotone constraints applied to arrears_balance (positive) and tenure_months (negative) to enforce domain logic.
LightGBM 4.1.0 35% WEIGHT
num_leaves=63 · min_data_in_leaf=20 · feature_fraction=0.8 · bagging_fraction=0.9 · bagging_freq=5 · lambda_l1=0.05 · lambda_l2=0.1 · learning_rate=0.05 · num_iterations=1000
Leaf-wise growth handles high-cardinality benefit code features natively. GOSS sampling retains informative high-gradient samples — critical given 18:82 class imbalance.
CatBoost 1.2.5 25% WEIGHT
depth=8 · l2_leaf_reg=3.0 · iterations=600 · learning_rate=0.06 · cat_features=['benefit_code','employment_type','tenure_band','property_type']
Handles categorical tenure and benefit codes without manual encoding — target statistics via ordered boosting prevent target leakage. Symmetric trees enable faster inference. Provides diversity through different categorical handling vs XGB/LGBM.
⓶ L2 — Stacking Meta-Learner CALIBRATED
Logistic Regression (Isotonic Calibrated) META
C=1.0 · penalty='l2' · solver='lbfgs' · max_iter=1000 · CalibratedClassifierCV(method='isotonic', cv=5)
Receives out-of-fold probability predictions from all 3 base models as features. Learns optimal linear combination. Post-hoc isotonic regression calibration transforms raw probabilities to match empirical frequencies — critical for reliable risk scoring. Brier score: 0.087.
⓷ L3 — Post-Processing & Specialist Layers 3 COMPONENTS
Pre-Arrears LSTM (Attention) 90-DAY SIGNALAAP EQUIVALENT
hidden_size=128 · num_layers=2 · dropout=0.3 · seq_len=24mo · input_size=18 · attention=BahdanauAttention · optimizer=AdamW(lr=1e-3,wd=1e-4) · epochs=80
Time-series model on 24-month payment/benefit history per tenant. Runs independently on GREEN-band tenants only. Output (P_90d_onset) feeds early-warning band. Parallel signal — does NOT replace L1-L2 stack. Trained on 36,000 historical onset events.
Fairlearn ThresholdOptimizer POST-HOCGDPR COMPLIANT
constraint=DemographicParity · objective=BalancedAccuracy · sensitive=['uc_status','age_band','tenancy_type']
Adjusts per-group classification thresholds post-calibration. Does NOT retrain model. Results: 3 group thresholds τ = {0.38, 0.42, 0.47}. UK Equality Act 2010 §149 PSED compliance. DP delta ≤ 0.05.
SARIMA Arrears Volume Forecaster CASHFLOW
order=(2,1,1) · seasonal_order=(1,1,1,12) · trend='ct' · enforce_stationarity=True · information_criterion='aic'
Portfolio-level forecaster for aggregate arrears count. Feeds cashflow planning and manager briefing agent. Fitted on 5 years monthly data. MAPE: 4.2%.
